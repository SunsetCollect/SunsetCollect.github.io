一、Front-Matter

Front-Matter 是一种标记语言，用于在文件的开头定义元数据。

其中的内容是以 --- 包裹的，并且可以包含多个键值对。

键值对以 key: value 的形式出现，其中 key 是键，value 是对应的值。

注：键值对中的key是不区分大小写的，但是value区分大小写。

以下是预定义的Front-Matter键值对

配置选项 默认值 描述
title MarkDown 的文件标题 文章标题
date 文件创建时的日期时间 建立日期
update 文件更新时的日期时间 更新日期
comments true 开启文章的评论功能
layout config.default_layout 文章的布局
tags 标签(不适用于分页)
categories 分类(不适用于分页)
permalink null 覆盖文章的永久链接，永久链应该以 / 或 .html结尾
excerpt 纯文本的页面摘要。使用该插件格式化文本
disableNunjucks flase 启用时禁用Nunjucks标签和标签插件的渲染功能
lang 继承自_config.yml 设置语言以覆盖自动检测
pulished 对于 _posts 下的文章为 true，对于 _draft 下的文章为 false 文章是否发布

JSON Front-matter
除了 YAML 外，你也可以使用 JSON 来编写Front-matter，只要将 — 代换成 ;;; 即可

二、Markdown-标题

可以使用#号来表示标题，#号的数量就可以看出标题的级数。
